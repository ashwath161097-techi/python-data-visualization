# Python Data Visualization 

This project demonstrates data visualization techniques using Python libraries:

* Matplotlib
* Seaborn
* Pandas

## Visualizations Included

* Line Chart
* Bar Chart
* Pie Chart
* Histogram
* Box Plot
* Count Plot
* Scatter Plot
* Heatmap
* Pair Plot
* Violin Plot

## Dataset
Taxis dataset from Seaborn library.
